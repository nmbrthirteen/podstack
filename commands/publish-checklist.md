# /publish-checklist — Launch Manager

> You are the launch manager. You ensure nothing is missed before, during, and after publishing content.

---

## Trigger

User says "publish checklist", "ready to publish", or is about to upload content.

---

## Pre-Upload

### File Preparation
- [ ] **File name = keywords** (e.g., `topic-guest-company.mp4`)
- [ ] Video in correct format (shorts: 9:16, podcast: 16:9)
- [ ] Audio levels balanced and clear
- [ ] Thumbnail files ready (correct dimensions)

### Metadata
- [ ] **Title:** Under 60 chars, hook in first 40, keyword in first 3 words
- [ ] **Description:** Hook first line, guest attribution, hashtags, links
- [ ] **Keywords:** Extracted from transcript, under 500 chars
- [ ] **Chapters:** Timestamps in description (long-form only)
- [ ] **Tags:** Topic + audience tags applied
- [ ] **Thumbnail:** Uploaded (not auto-generated)

### Linking
- [ ] **Video cards:** Added at relevant timestamps
- [ ] **End screen:** ONE specific next video (hyper specific = more clicks)
- [ ] **Playlist:** Added to relevant playlist
- [ ] **Related video:** Set for shorts (link back to long-form)

### The Playlist Hack
```
1. Go to playlist → Play All
2. Copy the &list=PLxxxxxx code from URL
3. Append to ANY video URL
4. Result: your playlist shows on right side, next autoplay = your video
```

### Deep Links
- [ ] Create smart link via ytopen.app (mobile → opens YouTube app)
- [ ] Combine with playlist hack

---

## At-Publish

### Engagement Setup (Do While Unlisted)
- [ ] **Pin comment:** CTA or question pinned BEFORE publishing
- [ ] **Community tab:** Schedule post 15 min after video goes live

### Short-Form Specific
- [ ] **Keyword CTA in video:** "Comment [KEYWORD] for the full video"
- [ ] **ManyChat/DM automation:** Auto-reply with deep link
- [ ] **Related video:** Link back to long-form source

---

## Post-Publish (First 24 Hours)

- [ ] **Reply to every comment** → signals engagement to algorithm
- [ ] **Heart valuable comments**
- [ ] **Instagram Stories (3-part):**
  1. Reshare a reel from the content
  2. Thumbnail + title + deep link
  3. Screen recording showing chapters + deep link
- [ ] **Update link in bio** with deep link
- [ ] **Share in relevant communities**

---

## Day 3-4 Optimization

| Signal | Action |
|--------|--------|
| CTR below average | A/B test title OR thumbnail (never both) |
| High impressions, low clicks | Thumbnail problem — test thumbnail first |
| Low impressions | SEO problem — check tags and description |
| Good CTR, low view duration | Hook problem — check opening seconds |
| Everything good | Schedule remaining shorts |

### A/B Testing Rules
- ONE thing at a time
- Wait 72 hours first
- Measure CTR as primary metric

---

## Shorts Batch Distribution

1. Mass schedule all shorts (spread across the week)
2. Always set "Related Video" to link back to long-form
3. Add keyword CTA overlay in video
4. Vary posting times to find best windows
