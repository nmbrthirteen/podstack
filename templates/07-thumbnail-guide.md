## Format 2: Shorts Thumbnails (9:16)

### Layout
```
┌──────────────────┐
│                   │
│   [VISUAL]        │
│                   │
│  ┌──────────────┐ │
│  │ LINE 1       │ │
│  │ LINE 2       │ │
│  └──────────────┘ │
└──────────────────┘
```

### Text Rules
- **Line 1:** ALL CAPS, bold
- **Line 2:** ALL CAPS, accent color
- **Position:** Lower third, centered
- **Max:** 4-6 words total
