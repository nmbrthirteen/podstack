# [Your Show Name] — Content Workflow

## End-to-End Pipeline

```
TRANSCRIPT → MOMENT EXTRACTION → TITLES → DESCRIPTIONS → THUMBNAILS → PUBLISH
```

---

## Phase 1: Transcript Analysis

1. First read: flag potential moments (energy shifts, surprising statements, stories)
2. Categorize flagged moments by content type
3. Score each moment (standalone value, hook strength, relevance, quotability)
4. Select top moments based on episode length

---

## Phase 2: Moment Documentation

For each moment, capture:
- Timestamp range
- Key quote
- Why it works
- 2-3 title options
- Thumbnail text (both formats)

---

## Phase 3: Title Development

1. Generate options using formulas from `05-title-formulas.md`
2. Verify against rules (length, keywords, banned words)
3. Select best option
4. Check no duplicates with existing shorts

---

## Phase 4: Description Writing

- Shorts: hook + attribution + link + hashtags
- Long-form: summary + timestamps + guest links + hashtags

---

## Phase 5: Thumbnail Planning

- Two-line text for each format
- Layout specs for designer
- Guest photo requirements

---

## Phase 6: Quality Assurance

- [ ] All moments standalone
- [ ] Titles accurate and on-brand
- [ ] Descriptions complete
- [ ] Thumbnails readable at small size
- [ ] No duplicates
- [ ] Guest/company names correct

---

## Handoff Requirements

### To Video Editor
- Timestamp ranges
- Cut instructions
- Audio notes

### To Thumbnail Designer
- Text for each thumbnail
- Guest photo specs
- Layout template

### To Publishing
- Titles and descriptions
- Posting schedule
- Cross-promotion notes

---
