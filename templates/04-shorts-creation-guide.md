# [Your Show Name] — Shorts Creation Guide

## Specifications

| Aspect | Value |
|--------|-------|
| **Platform** | YouTube Shorts |
| **Length** | Under 60 seconds (ideally 30-45 sec) |
| **Aspect Ratio** | 9:16 (vertical) |
| **Source** | Extracted from long-form episodes |

---

## High-Value Moment Types

1. **Surprising Facts/Statistics** — Numbers that make viewers pause
2. **Counterintuitive Insights** — Challenges common assumptions
3. **Actionable Wisdom** — Advice they can use today
4. **Origin Stories** — The "aha" moment, early struggles, pivots
5. **Unconventional Tactics** — Against-the-grain strategies
6. **Future Predictions** — Bold visions
7. **Emotional Peaks** — Passionate statements, vulnerable admissions

---

## Selection Criteria

### INCLUDE moments that have:
- [ ] Clear standalone value (no context needed)
- [ ] Strong opening hook (first 3 seconds grab attention)
- [ ] Single focused idea (one concept, fully delivered)
- [ ] Natural ending point (doesn't feel cut off)
- [ ] Emotional resonance
- [ ] Quotable phrasing

### AVOID moments that:
- [ ] Require context to understand
- [ ] Reference other parts of the conversation
- [ ] Are too technical without payoff
- [ ] Ramble without clear point
- [ ] Have poor audio quality
- [ ] Contain sensitive/unverified claims

---

## Content Types

| Type | What It Is | Primary Angle |
|------|-----------|--------------|
| **Guest Story** | Key decision, pivot, hard moment | The decision + non-obvious reason |
| **Technical Insight** | How something works, why it matters | The "so what" |
| **Market / Landscape** | Maps space, compares, calls trends | The sharp take |
| **Business / Strategy** | Revenue, pricing, fundraising | The ugly number |
| **Hot Take / Opinion** | Challenges beliefs | The bold claim |
